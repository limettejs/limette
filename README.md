# Limette

The JavaScript framework for Web Components and Deno.

> This is WIP.

## Features

- Deno on server
- Web Components (with Lit) on browser
- Island-based architecture
- File system routing
- Tailwind support

## Create a new Limette project

Run this command:

```
deno run -Ar jsr:@limette/init
```

## Start a project

```
deno task start
```
